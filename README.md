# hernansilva.com

Personal website for Hernan Silva. Single static page.

## Files
- `index.html` — the entire site (HTML, CSS, and JS in one file)

## How it deploys
This repo is connected to Cloudflare Pages. Every push to the `main` branch
automatically rebuilds and deploys the live site at https://hernansilva.com.

## Editing
Edit `index.html`, commit, and push. Cloudflare deploys within about a minute.
